# Klarna assignment

This assignment was solved in few hours, with some distractions on weekends.

To run the solution simple run `make` to compile, test and dialyze the code:

```shell
> make
./rebar3 compile
...
./rebar3 eunit
...
./rebar3 dialyzer
...
```
